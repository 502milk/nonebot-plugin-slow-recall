# nonebot-plugin-slow-recall

基于 NoneBot2 与 OneBot V11 的群聊慢速模式与延迟撤回插件。

本插件用于在群聊中按群独立设置发言频率限制，或让指定成员/全体成员发送的消息在指定时间后自动撤回。适合临时控场、防刷屏、限流讨论、消息阅后即焚等场景。

## 功能特性

- 支持群聊内独立配置规则，不同群互不影响
- 支持对全体成员或指定成员设置规则
- 支持慢速模式：限制 60 秒内允许发送的消息数量
- 慢速超限后可选择自动撤回消息或临时禁言
- 支持延迟撤回：消息发送后等待指定秒数自动撤回
- 支持查看慢速规则、延迟撤回规则和全部规则
- 规则自动持久化到 JSON 文件，重启后仍然生效
- 支持 NoneBot 超级用户、群主、群管理员管理规则

## 适用环境

- NoneBot2
- OneBot V11 适配器
- 支持 `delete_msg` 与 `set_group_ban` 的 OneBot 实现

## 安装

### 方式作为插件包加载

如果你已经将本项目放在 Python 可导入路径中，可直接加载插件模块。

## 加载插件

在 NoneBot 项目的入口文件中加载：

```python
import nonebot

nonebot.load_plugin("nonebot_plugin_slow_recall")
```

也可以在 NoneBot 配置的插件列表中加入：

```python
plugins = ["nonebot_plugin_slow_recall"]
```

实际加载方式请以你的 NoneBot 项目结构为准。

## 配置项

配置项写入 NoneBot 项目的 `.env` 或对应环境配置文件。

| 配置项 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `slow_recall_data_file` | `str` | `data/nonebot_plugin_slow_recall/config.json` | 规则数据保存路径 |
| `slow_recall_allow_superuser` | `bool` | `true` | 是否允许 NoneBot 超级用户管理规则 |
| `slow_recall_allow_group_admin` | `bool` | `true` | 是否允许群管理员管理规则 |
| `slow_recall_allow_group_owner` | `bool` | `true` | 是否允许群主管理规则 |

示例：

```env
slow_recall_data_file=data/nonebot_plugin_slow_recall/config.json
slow_recall_allow_superuser=true
slow_recall_allow_group_admin=true
slow_recall_allow_group_owner=true
```

数据文件路径如果是相对路径，会相对于机器人运行目录解析。

## 权限说明

只有满足以下任意条件的用户可以设置、关闭或查看规则：

- NoneBot 超级用户，并且 `slow_recall_allow_superuser=true`
- 群主，并且 `slow_recall_allow_group_owner=true`
- 群管理员，并且 `slow_recall_allow_group_admin=true`

普通群成员发送管理命令不会生效。

## 使用方法

所有命令均在群聊中发送。

### 慢速模式

慢速模式用于限制成员发言频率。默认统计窗口为 60 秒。

#### 对全体成员开启慢速模式

```text
慢速 全体 5 撤回
```

含义：每名成员 60 秒内最多发送 5 条消息，超过后自动撤回超限消息。

```text
慢速 全体 3 禁言
```

含义：每名成员 60 秒内最多发送 3 条消息，超过后临时禁言到当前统计窗口结束。

#### 对指定成员开启慢速模式

```text
慢速 @用户 2 撤回
```

含义：被 @ 的成员 60 秒内最多发送 2 条消息，超过后自动撤回。

```text
慢速 @用户 1 禁言
```

含义：被 @ 的成员 60 秒内最多发送 1 条消息，超过后临时禁言。

#### 省略处理方式

```text
慢速 全体 5
```

未填写 `撤回` 或 `禁言` 时，默认使用 `撤回`。

#### 小于 1 的慢速值

慢速值支持小数。小于 1 时表示更长间隔内允许 1 条消息。

例如：

```text
慢速 全体 0.5 撤回
```

含义：每名成员约 120 秒内最多发送 1 条消息。

### 关闭慢速模式

#### 关闭全体慢速

```text
慢速关闭 全体
```

也支持：

```text
慢速停止 全体
```

```text
慢速关 全体
```

#### 关闭指定成员慢速

```text
慢速关闭 @用户
```

### 查看慢速规则

查看当前群全部慢速规则：

```text
慢速列表
```

或：

```text
慢速列表 全体
```

查看指定成员慢速规则：

```text
慢速列表 @用户
```

## 延迟撤回

延迟撤回用于让目标成员发送的消息在指定时间后自动撤回。

### 对全体成员开启延迟撤回

```text
延迟撤回 全体 3秒
```

含义：全体成员发送的消息会在 3 秒后自动撤回。

也可以省略单位：

```text
延迟撤回 全体 3
```

### 对指定成员开启延迟撤回

```text
延迟撤回 @用户 0.5秒
```

含义：被 @ 的成员发送的消息会在 0.5 秒后自动撤回。

支持的时间单位包括：

- `秒`
- `秒钟`
- `s`
- `sec`
- `secs`
- `second`
- `seconds`

插件内部最小撤回延迟为 0.2 秒。设置小于该值时，会按 0.2 秒处理。

### 关闭延迟撤回

将延迟时间设置为 0 即可关闭对应规则。

关闭全体延迟撤回：

```text
延迟撤回 全体 0
```

关闭指定成员延迟撤回：

```text
延迟撤回 @用户 0
```

### 查看延迟撤回规则

查看当前群全部延迟撤回规则：

```text
延迟撤回列表
```

或：

```text
延迟撤回列表 全体
```

查看指定成员延迟撤回规则：

```text
延迟撤回列表 @用户
```

## 查看全部规则

查看当前群所有慢速模式和延迟撤回规则：

```text
规则列表
```

也可以发送：

```text
全部列表
```

## 规则优先级

当同一群内同时存在全体规则和指定成员规则时，指定成员规则优先。

例如当前群设置了：

```text
慢速 全体 5 撤回
慢速 @用户 1 禁言
```

则普通成员使用全体规则，指定成员使用自己的单独规则。

延迟撤回规则同理。

## 机器人权限要求

为了让插件正常执行操作，请确保机器人在群内拥有足够权限：

- 自动撤回需要机器人能够调用 OneBot 的 `delete_msg` 接口
- 禁言成员需要机器人是群管理员或群主，并能调用 `set_group_ban` 接口
- 机器人通常无法撤回超出平台限制时间的旧消息
- 机器人无法禁言群主或权限高于自己的管理员

## 数据保存

插件启动时会读取规则数据文件；设置或删除规则后会自动写入文件。

默认数据文件：

```text
data/nonebot_plugin_slow_recall/config.json
```

数据格式由插件自动维护，一般不需要手动编辑。

## 注意事项

- 本插件只处理群聊消息，不处理私聊消息
- 管理命令本身不会被已授权管理者的规则处理
- 如果 OneBot 实现不支持撤回或禁言接口，对应操作会失败并写入日志
- 延迟撤回会在机器人进程运行期间调度；机器人重启后，重启前已经排队但尚未执行的撤回任务不会恢复
- 慢速模式的计数窗口保存在内存中，机器人重启后会重新开始统计，但规则配置仍会保留
